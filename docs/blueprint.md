# **App Name**: ResumeAI

## Core Features:

- Resume Template Selection: Allow users to choose from a variety of resume templates, including classic, modern, and creative designs.
- AI-Powered Content Generation: Leverage AI to generate suggested content for resume sections like summaries, skills, and experience descriptions. Use a tool to determine when it is appropriate to incorporate information in the description.
- ATS Checker: Implement an ATS (Applicant Tracking System) checker that analyzes the resume content and formatting, providing feedback on how well it will perform in various ATS systems.
- Multi-Format Export: Enable users to export their resumes in multiple formats, such as PDF and DOCX, ensuring compatibility with various application systems and recruiter preferences.
- Real-Time Resume Preview: Display a real-time preview of the resume as the user edits their information, providing immediate visual feedback on how the final product will look.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to convey professionalism and competence.
- Background color: Light gray (#F0F2F5) to provide a clean and unobtrusive backdrop.
- Accent color: Teal (#009688) to highlight important elements like headings and call-to-action buttons.
- Headline font: 'Space Grotesk' sans-serif font for headlines, for a clean, techy feel.
- Body font: 'Inter' sans-serif font for body text, for maximum readability.
- Use modern, professional icons from Lucide or Font Awesome for visual cues, emphasizing clarity and ease of understanding.
- Maintain a clean, single-column layout for easy readability, ensuring the resume is ATS-friendly and visually appealing.
- Implement subtle animations, such as fade-ins and slide-ins, when switching between sections or generating content to provide a smooth user experience.
